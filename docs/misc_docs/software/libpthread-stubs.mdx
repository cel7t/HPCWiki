## Software - libpthread-stubs


#### Description
The libpthread-stubs package provides weak aliases for pthread functions not provided in libc or otherwise available by default.

#### Website
https://xcb.freedesktop.org/

#### Variants installed

1. libpthread-stubs 0.4

    - Compiler: gcc 10.2.0
    - Architecture: zen2

    Spack Command
    ```bash
    $ spack load libpthread-stubs/pwngdre
    ```

2. libpthread-stubs 0.4

    - Compiler: gcc 10.2.0
    - Architecture: zen2

    Spack Command
