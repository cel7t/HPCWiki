## Software - zstd


#### Description
Zstandard, or zstd as short version, is a fast lossless compression algorithm, targeting real-time compression scenarios at zlib-level and better compression ratios.

#### Website
https://facebook.github.io/zstd/

#### Variants installed

1. zstd 1.5.2

    - Compiler: gcc 8.5.0
    - Architecture: zen

    Spack Command
    ```bash
    $ spack load zstd/2omt76e
    ```

2. zstd 1.5.2

    - Compiler: gcc 11.2.0
    - Architecture: zen2

    Spack Command
