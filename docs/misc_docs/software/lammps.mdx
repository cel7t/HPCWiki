## Software - lammps


#### Description
LAMMPS stands for Large-scale Atomic/Molecular Massively Parallel Simulator. This package uses patch releases, not stable release. See https://github.com/spack/spack/pull/5342 for a detailed discussion.

#### Website
https://lammps.sandia.gov/

#### Variants installed

1. lammps 20200721

    - Compiler: gcc 10.2.0
    - Architecture: zen2
    - Supports MPI
    - Supports OpenMP

    Spack Command
    ```bash
    $ spack load lammps/goydecf
    ```

2. lammps 20200721

    - Compiler: aocc 2.3.0
    - Architecture: zen2
    - Supports MPI
    - Supports OpenMP

    Spack Command
    ```bash
    $ spack load lammps/dnp45yu
    ```

3. lammps 20210310

    - Compiler: gcc 10.2.0
    - Architecture: zen2
    - Supports MPI
    - Supports OpenMP

    Spack Command
    ```bash
    $ spack load lammps/oyb4fw3
    ```
