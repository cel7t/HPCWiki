## Software - charmpp


#### Description
Charm++ is a parallel programming framework in C++ supported by an adaptive runtime system, which enhances user productivity and allows programs to run portably from small multicore computers (your laptop) to the largest supercomputers.

#### Website
https://charmplusplus.org

#### Variants installed

1. charmpp 6.10.2

    - Compiler: aocc 2.3.0
    - Architecture: zen2

    Spack Command
    ```bash
    spack load charmpp/ydt4csz
    ```

2. charmpp 6.10.2

    - Compiler: gcc 10.2.0
    - Architecture: zen2

    Spack Command

    spack load charmpp/rcdhh4j
