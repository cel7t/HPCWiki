## Software - py-scipy


#### Description
SciPy (pronounced "Sigh Pie") is a Scientific Library for Python. It provides many user-friendly and efficient numerical routines such as routines for numerical integration and optimization.

#### Website
https://www.scipy.org/

#### Variants installed

1. py-scipy 1.6.0

    - Compiler: gcc 10.2.0
    - Architecture: zen2
