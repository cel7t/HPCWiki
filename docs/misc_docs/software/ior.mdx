## Software - ior


#### Description
The IOR software is used for benchmarking parallel file systems using POSIX, MPI-IO, or HDF5 interfaces.

#### Website
https://github.com/hpc/ior

#### Variants installed

1. ior 3.3.0

    - Compiler: gcc 10.3.0
    - Architecture: zen2

    Spack Command
    ```bash
    spack load ior/6mb6cw5
    ```
