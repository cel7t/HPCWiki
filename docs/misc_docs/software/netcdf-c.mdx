## Software - netcdf-c


#### Description
NetCDF (network Common Data Form) is a set of software libraries and machine-independent data formats that support the creation, access, and sharing of array-oriented scientific data. This is the C distribution.

#### Website
https://www.unidata.ucar.edu/software/netcdf

#### Variants installed

1. netcdf-c 4.7.4

    - Compiler: aocc 2.3.0
    - Architecture: zen2
    - Supports MPI

    Spack Command
    ```bash
    $ spack load netcdf-c/qvhngtp
    ```

2. netcdf-c 4.8.0

    - Compiler: gcc 10.2.0
    - Architecture: zen2
    - Supports MPI

    Spack Command
    ```bash
    $ spack load netcdf-c/n6jvy32
    ```

3. netcdf-c 4.8.1

    - Compiler: gcc 9.4.0
    - Architecture: zen2
    - Supports MPI

    Spack Command
