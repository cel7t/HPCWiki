## Software - nccl


#### Description
Optimized primitives for collective multi-GPU communication.

#### Website
https://github.com/NVIDIA/nccl

#### Variants installed

    - Compiler: gcc 8.5.0
    - Architecture: zen
    - Supports CUDA
