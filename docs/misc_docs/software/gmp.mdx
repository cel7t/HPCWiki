## Software - gmp


#### Description
GMP is a free library for arbitrary precision arithmetic, operating on signed integers, rational numbers, and floating-point numbers.

#### Website
https://gmplib.org

#### Variants installed

1. gmp 6.2.1

    - Compiler: gcc 8.5.0
    - Architecture: zen

    Spack Command
    ```bash
    $ spack load gmp/d6uhs3y
    ```

2. gmp 6.2.1

    - Compiler: gcc 11.2.0
    - Architecture: zen2

    Spack Command
