## Software - libffi


#### Description
The libffi library provides a portable, high level programming interface to various calling conventions. This allows a programmer to call any function specified by a call interface description at run time.

#### Website
https://sourceware.org/libffi/

#### Variants installed

1. libffi 3.4.2

    - Compiler: aocc 3.2.0
    - Architecture: zen2

    Spack Command
    ```bash
    $ spack load libffi/za2bup2
    ```

2. libffi 3.4.2

    - Compiler: gcc 8.5.0
    - Architecture: zen

    Spack Command

3. libffi 3.4.2

    - Compiler: gcc 11.2.0
    - Architecture: zen2
