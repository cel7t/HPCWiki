## Software - kbproto


#### Description
X Keyboard Extension. This extension defines a protcol to provide a number of new capabilities and controls for text keyboards.

#### Website
https://cgit.freedesktop.org/xorg/proto/kbproto

#### Variants installed

1. kbproto 1.0.7

    - Compiler: gcc 10.2.0
    - Architecture: zen2

    Spack Command
    ```bash
    $ spack load kbproto/rh2w75l
    ```

2. kbproto 1.0.7

    - Compiler: gcc 10.2.0
    - Architecture: zen2

    Spack Command
