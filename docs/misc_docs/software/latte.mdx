## Software - latte


#### Description
Open source density functional tight binding molecular dynamics.

#### Website
https://github.com/lanl/latte

#### Variants installed

1. latte 1.2.2

    - Compiler: aocc 2.3.0
    - Architecture: zen2
    - Supports MPI

    Spack Command
    ```bash
    $ spack load latte/veyapsf
    ```

2. latte 1.2.2

    - Compiler: gcc 10.2.0
    - Architecture: zen2
    - Supports MPI

    Spack Command
    ```bash
    $ spack load latte/nepxvwj
    ```
