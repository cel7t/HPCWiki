## Software - netlib-scalapack


#### Description
ScaLAPACK is a library of high-performance linear algebra routines for parallel distributed memory machines

#### Website
https://www.netlib.org/scalapack/

#### Variants installed

1. netlib-scalapack 2.1.0

    - Compiler: gcc 10.2.0
    - Architecture: zen2

    Spack Command
    ```bash
    spack load netlib-scalapack/ilxmkmp
    ```

2. netlib-scalapack 2.1.0

    - Compiler: gcc 11.2.0
    - Architecture: zen2

    Spack Command

    spack load netlib-scalapack/fbvwuhz

    - Compiler: gcc 11.2.0
    - Architecture: zen2
