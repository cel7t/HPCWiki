## Software - py-astunparse


#### Description
An AST unparser for Python. This is a factored out version of unparse found in the Python source distribution; under Demo/parser in Python 2 and under Tools/parser in Python 3.

#### Website
https://pypi.org/project/astunparse/

#### Variants installed

1. py-astunparse 1.6.3

    - Compiler: gcc 9.3.0
    - Architecture: zen2

    Spack Command
    ```bash
    $ spack load py-astunparse/x2xjvgh
    ```
