## Software - opencv


#### Description
OpenCV (Open Source Computer Vision Library) is an open source computer vision and machine learning software library.

#### Website
https://opencv.org/

#### Variants installed

1. opencv 4.1.2

    - Compiler: gcc 10.2.0
    - Architecture: zen2

    Spack Command
    ```bash
    spack load opencv/rfvktyl
    ```

2. opencv 4.5.1

    - Compiler: gcc 10.2.0
    - Architecture: zen2
    - Supports MPI
    - Supports CUDA

    Spack Command
    ```bash
    spack load opencv/qjydrqt
    ```
