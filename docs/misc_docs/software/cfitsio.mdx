## Software - cfitsio


#### Description
CFITSIO is a library of C and Fortran subroutines for reading and writing data files in FITS (Flexible Image Transport System) data format.

#### Website
https://heasarc.gsfc.nasa.gov/fitsio/

#### Variants installed

1. cfitsio 4.1.0

    - Compiler: gcc 11.2.0
    - Architecture: zen2

    Spack Command
    ```bash
    $ spack load cfitsio/foophid
    ```
