## Software - petsc


#### Description
PETSc is a suite of data structures and routines for the scalable (parallel) solution of scientific applications modeled by partial differential equations.

#### Website
https://www.mcs.anl.gov/petsc/index.html

#### Variants installed

1. petsc 3.14.1

    - Compiler: gcc 10.2.0
    - Architecture: zen2
    - Supports MPI
