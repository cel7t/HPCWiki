## Software - libxau


#### Description
The libXau package contains a library implementing the X11 Authorization Protocol. This is useful for restricting client access to the display.

#### Website
https://cgit.freedesktop.org/xorg/lib/libXau/

#### Variants installed

1. libxau 1.0.8

    - Compiler: gcc 10.2.0
    - Architecture: zen2

    Spack Command
    ```bash
    spack load libxau/kcgnwvl
    ```
