## Software - fftw


#### Description
Zstandard, or zstd as short version, is a fast lossless compression algorithm, targeting real-time compression scenarios at zlib-level and better compression ratios.

#### Website
https://facebook.github.io/zstd/

#### Variants installed

1. fftw 2.1.5

    - Compiler: gcc 11.2.0
    - Architecture: zen2
    - Supports MPI
    - Supports OpenMP

    Spack Command
    ```bash
    $ spack load fftw/y6jz5zg
    ```

2. fftw 3.3.10

    - Compiler: aocc 3.2.0
