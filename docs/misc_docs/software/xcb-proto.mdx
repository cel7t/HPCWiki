## Software - xcb-proto


#### Description
xcb-proto provides the XML-XCB protocol descriptions that libxcb uses to generate the majority of its code and API.

#### Website
https://xcb.freedesktop.org/

#### Variants installed

1. xcb-proto 1.14.1

    - Compiler: gcc 10.2.0
    - Architecture: zen2

    Spack Command
    ```bash
    spack load xcb-proto/2mrdcs4
    ```

2. xcb-proto 1.14.1

    - Compiler: gcc 10.2.0
    - Architecture: zen2

    Spack Command
    ```bash
    spack load xcb-proto/ap2lnc2
    ```
