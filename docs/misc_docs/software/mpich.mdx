## Software - mpich


#### Description
MPICH is a high performance and widely portable implementation of the Message Passing Interface (MPI) standard.

#### Website
https://www.mpich.org

#### Variants installed

1. mpich 3.3.2

    - Compiler: gcc 9.3.0
    - Architecture: zen2

    Spack Command
    ```bash
    $ spack load mpich/arbjh2r
    ```

2. mpich 3.3.2

    - Compiler: gcc 10.2.0
    - Architecture: zen2

    Spack Command
    ```bash
    $ spack load mpich/nrxxjej
    ```
